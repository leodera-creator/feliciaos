# FeliciaOS — Autonomous Web3 Intelligence Agent

Built from Anambra State, Nigeria by Pascal Chidera (@chipasleo)
No laptop. No CS degree. No capital. Just a phone.

## What FeliciaOS Does
A 5-layer autonomous crypto and Web3 intelligence agent powered by LLaMA 3.3 70B via Groq.

- Layer 1: Market Intelligence Oracle
- Layer 2: Web3 Community Growth Engine
- Layer 3: On-Chain Narrative Builder
- Layer 4: Crypto Confessions Engine
- Layer 5: DeFi Education & Strategy Layer

## Stack
- Python 3 + Groq API (free)
- LLaMA 3.3 70B
- Built entirely on Android via Termux + Ubuntu proot

## Run It
pip install groq python-dotenv schedule
python3 felicia_brain.py --layer market_scan
